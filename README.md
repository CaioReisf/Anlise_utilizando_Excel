# 📊 Análise de Vendas Comerciais utilizando Excel

## Sobre o Projeto

Este projeto consiste em uma análise exploratória de dados de vendas desenvolvida utilizando exclusivamente Microsoft Excel.

O objetivo foi analisar o desempenho comercial de uma empresa fictícia, identificando padrões de vendas, produtos mais relevantes, comportamento regional e oportunidades de melhoria através dos dados.

---

# Contexto de Negócio

A Empresa possui uma grande quantidade de registros de vendas, porém necessita transformar esses dados em informações estratégicas para auxiliar na tomada de decisão.

A análise busca responder:

- Quais produtos possuem maior impacto no faturamento?
- Quais regiões apresentam melhor desempenho?
- Como as vendas evoluem ao longo do tempo?
- Quais categorias possuem maior participação na receita?

---

# Objetivos

- Avaliar o desempenho geral das vendas;
- Identificar produtos e categorias mais relevantes;
- Analisar faturamento por período;
- Comparar desempenho entre regiões;
- Criar um dashboard executivo para acompanhamento dos indicadores.

---

# Dataset

A base utilizada contém 600 registros de vendas fictícias.

Principais informações:

| Campo | Descrição |
|-|-|
| ID Venda | Identificador da venda |

| Região | Região da venda |
| Quantidade | Quantidade vendida |
| Preço Unitário | Valor individual |
| Desconto | Desconto aplicado |
| Receita | Receita final |

# Principais Insights

Durante a análise foram identificados:

- A categoria de Eletrônicos apresentou maior participação no faturamento;
- Alguns produtos possuem alta quantidade vendida, mas menor impacto financeiro;
- Determinados períodos apresentaram maior concentração de vendas;
- Algumas regiões possuem maior contribuição para a receita total.



# Recomendações

Com base nos dados analisados:

- A empresa deve priorizar investimentos nos produtos com maior participação na receita;
- Regiões com baixo desempenho podem receber campanhas específicas;
- Produtos com alta venda e baixa receita devem ter sua estratégia revisada.



# Conclusão

O projeto demonstrou como ferramentas do Excel podem ser utilizadas para transformar dados brutos em informações estratégicas.

Através da análise e visualização dos dados foi possível identificar padrões comerciais e criar uma visão mais clara do desempenho da empresa.
